# PowerBI-SQL-Car-Sales-Analysis

## Overview
Interactive car sales analytics dashboard built using SQL and Microsoft Power BI to analyze sales performance, revenue, vehicle pricing, dealership performance, and key business KPIs. The dashboard provides valuable insights into sales trends, customer preferences, and regional performance to support data-driven business decisions.

## Tools Used
- SQL
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

## What I Did
- Queried and analyzed car sales data using SQL.
- Cleaned and transformed the dataset using Power Query.
- Created DAX measures for key business KPIs.
- Designed an interactive Power BI dashboard with dynamic filters and visualizations.
- Analyzed sales performance across vehicle body styles, colors, dealers, and regions.

## Dashboard Features
- Year-to-Date (YTD) Sales
- Month-to-Date (MTD) Sales
- Average Vehicle Price
- Cars Sold
- Weekly Sales Trend
- Sales by Vehicle Body Style
- Sales by Vehicle Color
- Dealer Region Analysis
- Company-wise Sales Performance
- Interactive Filters (Body Style, Dealer, Transmission, Engine)

## Key KPIs
- YTD Total Sales
- MTD Total Sales
- Average Vehicle Price
- Cars Sold
- Year-over-Year (YoY) Growth
- Month-over-Month (MoM) Performance
